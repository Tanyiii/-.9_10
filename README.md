# Временные сложности:
Блочная сортировка:

Лучший: O(n + k)

Средний: O(n + k)

Худший: O(n²)

Блинная сортировка:

Всегда: O(n²)

Сортировка бусинами:

Теоретическая: O(n)

Практическая: O(n × m), где m - максимальное значение

Поиск скачками:

O(√n)

Экспоненциальный поиск:

O(log n)

Тернарный поиск:

O(log₃ n) ≈ O(log n)
